# Stack Map

## Overview
- **Type:** Frontend Single Page Application (SPA)
- **Language:** TypeScript
- **Framework:** React 19, Vite
- **Styling:** Tailwind CSS v4, Lucide React (icons)
- **State Management:** React hooks (`useState`, `useLocalStorage`)

## Dependencies
- `react`, `react-dom`
- `@tailwindcss/vite`, `tailwindcss`
- `vite`
- `xlsx` (for parsing spreadsheets)

## Scripts
- `npm run dev` - Starts development server
- `npm run build` - Builds for production

## Infrastructure
- Hosted static site (no dedicated backend)
- No database currently configured (all data bundled or cached in local storage)
