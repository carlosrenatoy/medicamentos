🚀 Como Publicar seu App (Modo Fácil)

Não se preocupe! Vamos fazer isso juntos, passo a passo, como uma receita de bolo.

O que vamos fazer (Resumo)
  1. Preparar o pacote: Juntar tudo numa pasta pronta pra viagem.
  2. Configurar o Servidor (VPS): Instalar o "sistema" que vai exibir seu site.
  3. Enviar os arquivos: Mandar o pacote do seu computador para o servidor (usando um programa visual).
  4. Apontar o Domínio: Dizer para a internet onde seu site mora.

Parte 1: Preparar o Pacote 📦

No seu computador (onde está o projeto):

  1. Abra a pasta do projeto ipass-scut.
  2. Procure o arquivo chamado criar_pacote_deploy.bat.
  3. Dê dois cliques nele.
  4. Uma tela preta vai aparecer e sumir rápido. Isso é normal!
  5. Resultado: Vai aparecer uma nova pasta chamada dist. Ela contém seu site pronto.

Parte 2: Configurar o Servidor (VPS) 🖥️

Acesso ao Terminal

Você precisa acessar sua VPS. Se você usa Windows, abra o PowerShell.

ssh root@SEU_IP_DA_VPS

(Substitua SEU_IP_DA_VPS pelo número, ex: 142.93.100.55). Ele vai pedir a senha. Digite e dê Enter (a senha não aparece enquanto você digita).

Instalação Automática

Dentro da tela preta da VPS, copie e cole este comando mágico (todo de uma vez) e dê Enter:

apt update && apt install nginx -y && ufw allow 'Nginx Full'

Espere terminar. Se aparecer algo perguntando [Y/n], digite Y e Enter.

Teste: Abra seu navegador (Chrome/Edge) e digite o IP da sua VPS na barra de endereço. Se aparecer uma página dizendo "Welcome to nginx", Parabéns! O servidor está vivo! 🎉

Parte 3: Enviar os Arquivos (Modo Visual) 📤

Usando o FileZilla

Para não usar comandos difíceis, vamos usar um programa gratuito chamado FileZilla (ou WinSCP).

  1. Baixe e instale o FileZilla Client.
  2. Abra o FileZilla.
  3. Preencha os campos lá em cima:
    • Host: sftp://SEU_IP_DA_VPS (coloque sftp:// antes do IP)
    • Usuário: root
    • Senha: A senha da sua VPS
    • Porta: 22
  4. Clique em Conexão Rápida.
Arrastando os Arquivos

Agora você verá duas colunas:

  • Esquerda: Seu computador.
  • Direita: O servidor VPS.
  1. Na direita (Servidor), navegue até a pasta: /var/www/html/
  2. Apague o arquivo index.nginx-debian.html que estiver lá (se tiver).
  3. Na esquerda (Seu PC), entre na pasta dist que criamos no Passo 1.
  4. Selecione TUDO que está dentro da pasta dist (index.html, css, js, assets).
  5. Arraste para o lado direito.
Pronto! Acesse o IP da VPS no navegador novamente. Seu site IPASS SCUT deve aparecer!

Parte 4: O Domínio (O Nome do Site) 🌐

Isso é feito no site onde você comprou o domínio (não é na VPS).

  1. Entre no site (Registro.br, GoDaddy, etc).
  2. Vá em "Gerenciar DNS" ou "Editar Zona".
  3. Clique em Adicionar Registro.
  4. Escolha Tipo: A.
  5. Nome: @ (ou deixe em branco).
  6. Valor/Destino: O IP da sua VPS.
  7. Salvar.

Demora um pouco (até 24h) para funcionar no mundo todo.
