# IPASS SCUT - Sistema de Passagem de Plantão Médico

Sistema web moderno para gerenciamento de passagens de plantão médico, facilitando a comunicação entre equipes durante trocas de turno.

## 🚀 Funcionalidades

- ✅ **Dashboard Interativo**: Visualização rápida de pacientes ativos, pendências e estatísticas
- ✅ **Registro de Passagens**: Formulário completo para documentar informações do paciente
- ✅ **Gestão de Pacientes**: Lista de pacientes em acompanhamento com busca e filtros
- ✅ **Histórico Completo**: Registro de todas as passagens realizadas
- ✅ **Armazenamento Local**: Dados salvos no navegador (localStorage)
- ✅ **Design Responsivo**: Interface adaptável para desktop, tablet e mobile
- ✅ **Tema Médico Profissional**: Design moderno com cores e animações suaves

## 📋 Informações Registradas

Para cada paciente, o sistema permite registrar:

- **Dados Básicos**: Nome, idade, leito/quarto, diagnóstico
- **Status Clínico**: Condição atual (estável, monitoramento, crítico, melhorando)
- **Sinais Vitais**: PA, FC, FR, temperatura, saturação, etc.
- **Pendências**: Exames, procedimentos e tarefas pendentes
- **Intercorrências**: Eventos importantes nas últimas 24h
- **Medicações**: Lista de medicamentos em uso
- **Observações**: Informações adicionais relevantes

## 🎨 Tecnologias

- **HTML5**: Estrutura semântica
- **CSS3**: Design system completo com variáveis CSS
- **JavaScript Vanilla**: Lógica da aplicação sem dependências
- **LocalStorage**: Persistência de dados no navegador

## 🚀 Como Usar

1. Abra o arquivo `index.html` em qualquer navegador moderno
2. Clique em "Nova Passagem" para registrar um paciente
3. Preencha as informações do formulário
4. Salve para adicionar à lista de pacientes ativos
5. Clique em qualquer paciente para ver detalhes completos
6. Use o botão "Histórico" para ver todas as passagens registradas

## 📱 Compatibilidade

- ✅ Chrome/Edge (versões recentes)
- ✅ Firefox (versões recentes)
- ✅ Safari (versões recentes)
- ✅ Dispositivos móveis (iOS/Android)

## 💾 Armazenamento de Dados

Os dados são armazenados localmente no navegador usando `localStorage`. Isso significa que:

- ✅ Funciona offline
- ✅ Não requer servidor ou banco de dados
- ⚠️ Dados são específicos do navegador/dispositivo
- ⚠️ Limpar cache do navegador remove os dados

## 🔒 Privacidade

Este sistema armazena todos os dados localmente no navegador do usuário. Nenhuma informação é enviada para servidores externos.
